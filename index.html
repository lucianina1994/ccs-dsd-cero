Selectores simples

Etiqueta simple o elemento

Son etiquetas HTML. Sirven para normalizar el estilo en todos los navegadores. Normalizar: Sobreescribir los estilos en todos los navegadores. Normalize.css es un archivo que se puede descargar, aqui ya esta la normalizacion.

h1 { color:rgb(10, 9, 94); } p { color:rgb(46, 13, 163); }

De Clase

Se indican con un .nombre-de-la-clase No pueden comenzar con numero Pueden comenzar con - o _ Pueden contener un : Ej: .title:first Pueden ser de una letra

Por ID

No se pueden repetir (No son reutilizables). Tienen especificidad muy alta. No es recomendable su uso.

Selectores compuestos

Agrupados

Basicamente es agrupar elementos html. No es recomendable su uso.

h1, h2, h3 { color:rgb(10, 9, 94); }

Combinados

aplica en todos los h1 que tengan clase title. No tiene una separacion de por medio.

h1.title { color:rgb(10, 9, 94); }

Decendientes

html es un arbol que tiene padres e hijo, como uno genialigico, body es padre de todos sus hijos (h1, p) Se usa cuando se quiere definir claramente una estructura, como por ejemplo si estoy haciendo un menu, generalmente los menus tienen items y los items sub-items.

HTML < body > < h1 >< /h1 > < p >< /p > < /body > CSS body p { color:blue; }

Con operador

operadores * selector universal: (selecciona absolutamente todo), no es buena idea usarlo, es mejor normalizar. > hijo directo: funciona si la relacion es solo de hijo directo. + hermano siguiente: se aplica solo a los siguientes hermanos de una determinada etiqueta ~ hermanos siguientes: por ejemplo, todos los divs que vengan luego de un p.

h1 ~ h2 { color:white; }

De atributo

Usan []

[attr="value"] { color:white; }
